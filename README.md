## Local Setup

```sh
$ git clone git@github.com:adhit7/webshop.git
```

```sh
$ cd webshop
```

```sh
$ npm install
```


```sh
$ cd frontend
```

```sh
$ npm install
```


## Run Locally

```sh
Create your own env file MONGO DB connection and JWT token
```

```sh
$ npm run dev
```
